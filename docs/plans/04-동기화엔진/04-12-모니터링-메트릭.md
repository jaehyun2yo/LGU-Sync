# 12. 모니터링 및 메트릭

### 12.1 실시간 메트릭

동기화 엔진의 상태를 실시간으로 파악하기 위한 메트릭을 수집한다.

```typescript
interface SyncMetrics {
  // === 파일 처리 메트릭 ===
  files: {
    totalDetected: number;         // 총 감지된 파일 수
    totalProcessed: number;        // 총 처리 완료 파일 수
    totalFailed: number;           // 총 실패 파일 수
    totalSkipped: number;          // 총 건너뛴 파일 수 (중복 등)
    currentQueueSize: number;      // 현재 큐 대기 수
    currentActiveJobs: number;     // 현재 진행 중 작업 수
  };

  // === 전송량 메트릭 ===
  transfer: {
    totalDownloadedBytes: number;  // 총 다운로드 바이트
    totalUploadedBytes: number;    // 총 업로드 바이트
    currentDownloadSpeedBps: number; // 현재 다운로드 속도
    currentUploadSpeedBps: number;   // 현재 업로드 속도
    averageDownloadSpeedBps: number; // 평균 다운로드 속도
    averageUploadSpeedBps: number;   // 평균 업로드 속도
  };

  // === 성공률 메트릭 ===
  rates: {
    downloadSuccessRate: number;   // 다운로드 성공률 (%)
    uploadSuccessRate: number;     // 업로드 성공률 (%)
    overallSuccessRate: number;    // 전체 성공률 (%)
    detectionCoverage: number;     // 감지 커버리지 (%)
  };

  // === 에러 메트릭 ===
  errors: {
    totalErrors: number;           // 총 에러 수
    networkErrors: number;         // 네트워크 에러 수
    authErrors: number;            // 인증 에러 수
    rateLimitErrors: number;       // 레이트 리밋 에러 수
    fileSystemErrors: number;      // 파일시스템 에러 수
    serverErrors: number;          // 서버 에러 수
    retryQueueSize: number;        // 재시도 큐 크기
    dlqSize: number;               // DLQ 크기
  };

  // === 엔진 상태 메트릭 ===
  engine: {
    state: string;                 // 현재 엔진 상태
    uptime: number;                // 가동 시간 (ms)
    circuitBreakerState: string;   // 서킷 브레이커 상태
    lastPollAt: number;            // 마지막 폴링 시각
    lastSnapshotAt: number;        // 마지막 스냅샷 시각
    lastIntegrityCheckAt: number;  // 마지막 무결성 검증 시각
    memoryUsageMB: number;         // 메모리 사용량 (MB)
  };

  // === 인증 메트릭 ===
  auth: {
    isAuthenticated: boolean;      // 인증 상태
    cookieExpiresAt: number;       // 쿠키 만료 시각
    reauthCount: number;           // 재인증 횟수
    lastAuthAt: number;            // 마지막 인증 시각
  };

  // 메트릭 수집 시각
  collectedAt: number;
}
```

```
┌──────────────────────────────────────────────────────────────┐

└──────────────────────────────────────────────────────────────┘

  ┌────────────────────┬───────────┬──────────────────────────┐
  │ 메트릭 카테고리     │ 수집 주기  │ 수집 방식               │
  ├────────────────────┼───────────┼──────────────────────────┤
  │ 파일 처리 카운터    │ 실시간    │ 이벤트 처리 시 즉시 갱신  │
  │ 전송 속도          │ 1초      │ 슬라이딩 윈도우 평균      │
  │ 성공률             │ 5초      │ 최근 100건 기반 계산      │
  │ 에러 카운터        │ 실시간    │ 에러 발생 시 즉시 갱신    │
  │ 엔진 상태          │ 5초      │ 상태 머신에서 수집        │
  │ 메모리 사용량       │ 30초     │ process.memoryUsage()   │
  │ 인증 상태          │ 30초     │ 쿠키 만료 시각 확인       │
  └────────────────────┴───────────┴──────────────────────────┘

  GUI 업데이트 주기: 1초 (IPC 이벤트 발행)
```

### 12.3 GUI 대시보드로 전달 방식 (IPC 이벤트)

Electron의 IPC(Inter-Process Communication)를 사용하여 메인 프로세스(동기화 엔진)에서 렌더러 프로세스(GUI)로 메트릭을 전달한다.

```
┌──────────────────────────────────────────────────────────────┐
│                  IPC 메트릭 전달 아키텍처                       │
└──────────────────────────────────────────────────────────────┘

  메인 프로세스 (Main Process)
  ┌───────────────────────────────────────────────┐
  │                                               │
  │  ┌──────────────┐    ┌───────────────────┐    │
  │  │ 동기화 엔진   │───▶│ MetricsCollector  │    │
  │  │ (SyncEngine) │    │                   │    │
  │  └──────────────┘    │ 1초 간격으로       │    │
  │                      │ 메트릭 수집        │    │
  │                      └────────┬──────────┘    │
  │                               │               │
  │                      ipcMain.emit()           │
  │                               │               │
  └───────────────────────────────┼───────────────┘
                                  │
                            IPC Channel
                          'sync:metrics'
                          'sync:progress'
                          'sync:error'
                          'sync:status'
                                  │
  렌더러 프로세스 (Renderer Process)
  ┌───────────────────────────────┼───────────────┐
  │                               │               │
  │                      ipcRenderer.on()         │
  │                               │               │
  │                      ┌────────▼──────────┐    │
  │                      │ MetricsStore      │    │
  │                      │ (React State)     │    │
  │                      └────────┬──────────┘    │
  │                               │               │
  │              ┌────────────────┼────────────┐  │
  │              │                │            │  │
  │              ▼                ▼            ▼  │
  │  ┌────────────────┐ ┌─────────────┐ ┌──────┐│
  │  │ 상태 대시보드   │ │ 파일 목록   │ │ 로그 ││
  │  │ (StatusPanel)  │ │ (FileList)  │ │(Log) ││
  │  └────────────────┘ └─────────────┘ └──────┘│
  │                                               │
  └───────────────────────────────────────────────┘
```

**IPC 채널 정의:**

```typescript
// 메인 → 렌더러 (단방향)
interface IPCChannels {
  // 전체 메트릭 업데이트 (1초 간격)
  'sync:metrics': SyncMetrics;

  // 개별 파일 진행률 (실시간)
  'sync:progress': DownloadProgress | UploadProgress;

  // 에러 알림 (발생 시 즉시)
  'sync:error': {
    type: string;
    message: string;
    details?: unknown;
    timestamp: number;
  };

  // 엔진 상태 변경 (상태 전이 시)
  'sync:status': {
    previousState: string;
    currentState: string;
    reason: string;
    timestamp: number;
  };

  // DLQ 알림 (항목 추가 시)
  'sync:dlq': {
    action: 'added' | 'removed' | 'cleared';
    entry?: DLQEntry;
    totalCount: number;
  };

  // 인증 상태 변경
  'sync:auth': {
    status: 'authenticated' | 'expired' | 'reauth_required' | 'reauth_success' | 'reauth_failed';
    message: string;
  };
}

// 렌더러 → 메인 (명령)
interface IPCCommands {
  // 엔진 제어
  'sync:start': void;
  'sync:pause': void;
  'sync:resume': void;
  'sync:stop': void;

  // DLQ 관리
  'sync:dlq:retry': { eventId: string };
  'sync:dlq:retryAll': void;
  'sync:dlq:delete': { eventId: string };
  'sync:dlq:deleteAll': void;

  // 설정 변경
  'sync:config:update': Partial<SyncEngineConfig>;

  // 수동 트리거
  'sync:trigger:snapshot': void;     // 수동 스냅샷 비교 실행
  'sync:trigger:integrity': void;    // 수동 무결성 검증 실행
}
```

```typescript
    if (!this.lastEmitted) {
      // 첫 전송: 전체 메트릭
      ipcMain.emit('sync:metrics', current);

      const delta = this.computeDelta(this.lastEmitted, current);
```

### 12.4 메트릭 로깅

메트릭은 콘솔 로그와 파일 로그에도 주기적으로 기록된다.

```typescript
// 5분 간격으로 메트릭 요약 로그 기록
setInterval(() => {
  const metrics = metricsCollector.collect();
  logger.info('=== 동기화 메트릭 요약 ===', {
    uptime: formatDuration(metrics.engine.uptime),
    engineState: metrics.engine.state,
    filesProcessed: metrics.files.totalProcessed,
    filesFailed: metrics.files.totalFailed,
    queueSize: metrics.files.currentQueueSize,
    downloadSpeed: formatBytes(metrics.transfer.averageDownloadSpeedBps) + '/s',
    successRate: metrics.rates.overallSuccessRate.toFixed(1) + '%',
    errors: metrics.errors.totalErrors,
    dlqSize: metrics.errors.dlqSize,
    memoryMB: metrics.engine.memoryUsageMB.toFixed(1),
    circuitBreaker: metrics.engine.circuitBreakerState,
  });
}, 300000);
```
