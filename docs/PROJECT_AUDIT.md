#### A9: SessionStart 훅

**왜 필요한가**: 세션 시작 시 자동으로 프로젝트 상태를 주입하여 에이전트가 즉시 작업에 투입될 수 있다.

**생성할 파일**: `.claude/settings.json` 수정

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Bash",
        "hooks": [
          {
            "type": "command",
            "command": "bash .claude/scripts/check-work-log.sh"
          }
        ]
      }
    ]
  }
}
```

> 참고: SessionStart 훅은 OMC의 글로벌 설정에서 관리. 프로젝트 레벨에서는 CLAUDE.md 세션 프로토콜로 대체 가능.
