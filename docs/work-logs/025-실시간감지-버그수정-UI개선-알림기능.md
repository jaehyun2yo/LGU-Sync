## 주요 결정사항

1. **감지 누락 완전 해결**: `consecutiveOldPages` 조기종료 로직을 완전 제거. 모든 페이지를 스캔하여 페이지 사이에 끼인 신규 항목 누락 방지 (MAX_POLL_PAGES=50 상한은 유지)
2. **디렉토리 경로 3단계 해결**: API `itemFolderFullpath` → DB `sync_folders.lguplus_folder_path` → "/" 폴백. folder-discovery에서 기존 폴더의 path도 업데이트하여 근본 원인 해결
3. **스캔 진행바 제거**: 사용자 요청에 따라 ScanProgressBar 컴포넌트 완전 제거. 대신 시작 중 단계별 진행 메시지로 대체
4. **시작 단계별 진행**: DetectionService.start() 6단계마다 EventBus → IPC → Renderer 전파. StatusPanel에서 RefreshCw 스피너와 함께 현재 단계 메시지 표시
5. **OS 알림**: Electron `Notification` API 사용. watchFolderIds로 필터링, folderId별 그룹화하여 operCode 요약을 body에 표시
