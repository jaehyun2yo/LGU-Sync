## 주요 결정사항
- **디렉토리 구조 유지 다운로드는 이미 구현되어 있음** — getPathSegments() + mkdir({recursive: true})로 원본 구조 재현 확인
- isPolling 플래그(boolean)로 동시성 제어 — mutex보다 단순하고 Node.js 싱글스레드에서 충분
- scan-progress 이벤트는 기존 sync:progress 패턴과 동일한 방식으로 일관성 유지
- buildFileTree는 Renderer 순수 함수로 구현하여 Main 프로세스 개입 없이 처리
